# Currencyops

Currencyops is a monitoring application that allows you to monitor currency exchange rates, stocks and cryptocurrencies.

## Usage
 `python3 main.py`

## Required Libraries
  To install all of the required libraries use:\
`pip3 install -r requirements.txt`
